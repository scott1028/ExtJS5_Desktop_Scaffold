# SSD Web Front-End by ExtJS
- How to start
    1. install nodejs
    2. git clone https://github.com/scott1028/ExtJS5_Desktop_Scaffold
    3. npm install
    4. grunt build:dev

- APIs List

API | Description
--- | ---
**grunt lift**      | To lift development server
**grunt halt**      | Force to stop development server by kill ps named grunt
